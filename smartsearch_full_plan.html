The LangGraph Agent Graph

4 agent nodes connected in a stateful cyclic graph. The Critic can loop back to Retriever if confidence is low — this is what makes it agentic, not just a pipeline.

🔍 QueryAnalyser
→
📚 Retriever
→
✍️ Synthesiser
→
🧐 Critic
↑ Critic loops back to Retriever with rewritten query if confidence < 0.7 (max 3 retries)

🔍 QueryAnalyser Agent

Receives raw user question. Classifies intent (factual / comparative / summarisation). Rewrites query for better semantic search. Selects which ChromaDB collection to search.

Tools: LLM call, query rewriter

📚 Retriever Agent

Calls Vector Service REST API with the rewritten query. Fetches top-k chunks. Scores relevance. If top chunk score < threshold, flags as "low confidence" for Critic to handle.

Tools: httpx → Vector Service /query endpoint

✍️ Synthesiser Agent

Receives retrieved chunks. Calls LLM with structured prompt: "Answer the question using ONLY these sources. Cite each source." Returns answer + list of source references.

Tools: LLM call (Claude / GPT-4o)

🧐 Critic Agent

Scores the answer on 3 dimensions: relevance, groundedness, completeness (0–1 each). If avg score < 0.7 and retry_count < 3, rewrites query and loops back to Retriever. Otherwise returns final answer.

Tools: LLM call, scoring rubric

LangGraph State Schema
from typing import List, Optional, TypedDict

class SearchState(TypedDict):
    # Input
    question:        str
    collection_name: str

    # Agent outputs
    rewritten_query: str
    intent:          str               # factual | comparative | summary
    documents:       List[str]
    sources:         List[str]
    generation:      str
    confidence:      float             # 0.0 – 1.0

    # Control flow
    retry_count:     int
    final_answer:    Optional[str]

1. Vector Service — ChromaDB REST wrapper
from fastapi import FastAPI
import chromadb

app = FastAPI()
client = chromadb.PersistentClient(path="/data/chroma")

@app.post("/query")
async def query(collection: str, query: str, top_k: int = 5):
    col = client.get_collection(collection)
    results = col.query(query_texts=[query], n_results=top_k)
    return {"documents": results["documents"][0],
            "distances": results["distances"][0]}
2. Critic Agent — confidence scoring
async def critic_node(state: SearchState) -> SearchState:
    prompt = f"""Score this answer 0.0–1.0 on 3 axes.
Question: {state['question']}
Answer:   {state['generation']}
Sources:  {state['documents']}

Return JSON: {{"relevance": 0.x, "groundedness": 0.x, "completeness": 0.x}}"""

    response = await llm.ainvoke(prompt)
    scores = json.loads(response.content)
    confidence = sum(scores.values()) / 3

    return {**state, "confidence": confidence,
            "retry_count": state["retry_count"] + 1}

# Conditional edge
def should_retry(state: SearchState) -> str:
    if state["confidence"] < 0.7 and state["retry_count"] < 3:
        return "retriever"   # loop back
    return "end"           # done
